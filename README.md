## Setup
To run this project, install it locally using npm:
```
$ cd ../atlas
$ npm install
```
To run concurrently (backend & frontend at the same time)
```
$ npm run dev
```
To run only backend 
```
$ npm run backend
```
To run only frontend
```
$ npm run frontend
```
